# DynoCue

> **Note:** This project is currently **in progress** and is **not yet functional**.

Documentation, Architecture and releases will be available at [dynocue.com](https://dynocue.com).

## Project Overview

DynoCue is an extensible control system intended for building repeatable shows and events. It provides a modular architecture for integrating various control elements, including audio, lighting, and other show control systems.

## Development

DynoCue is currently in development. There are currently no guarantees of stability or functionality.
